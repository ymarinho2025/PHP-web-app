Arquivos convertidos para PostgreSQL/Neon.

Substitua mantendo a estrutura:

public/*.php -> arquivos públicos do site
src/Controllers/db.php -> conexão PDO/Neon
src/Controllers/migrate.php -> migração via CLI
src/Controllers/login/*.php -> autenticação/registro/login
src/Controllers/image/*.php -> upload/listagem de imagens
src/Controllers/admin/*.php -> painel/admin

IMPORTANTE:
1. Configure a variável DATABASE_URL na hospedagem.
2. Rode a migração pelo terminal: php src/Controllers/migrate.php
3. Não deixe migrate.php acessível pela web.
4. No register.php foi removido o event.preventDefault(), pois ele impedia o cadastro.
5. Foram removidos usos de mysqli, bind_param, get_result, AUTO_INCREMENT e ENGINE=InnoDB.
